The equal interval classification creates data classes of equal size, with the amount in each class determined by the maker. For example, in this map, I chose to create 5 equal interval classes. This method is most useful for classifying continuous data sets and is also advantageous because it creates a legend that is easy to understand. A disadvantage of this method is that the data will likely end up falling into one or two categories, as can be seen in this map. The map with data classified using natural breaks looks similar to map whose data was classified as equal interval, however they are not necessarily always similar. Natural breaks classes are created to group similar data together, therefore maximizing the difference between classes. This classification method is more useful when grouping data that is unevenly distributed but not necessarily skewed. Like the quantile classification method one potential disadvantage is the possibility of a wide range of data values within a single class.
